# ObsSM
ALMA Log's State Machine Parser

#Compiling Process

* Download sources [https://github.com/javierivanov/ObsSM/archive/obssm-0.1.zip](obssm-0.1)
* Unzip files

```
unzip obssm-0.1.zip
```

* Enter to the folder

```
cd obssm-0.1/ObsSM/
```

* Compile using Maven

```
mvn package
```
